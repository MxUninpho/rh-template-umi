# @roothub/components

dumi 目前不支持 umi4 版本：https://github.com/umijs/dumi/issues/1119

暂不支持组件文档预览

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
